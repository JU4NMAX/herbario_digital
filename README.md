🌿 Herbario de Campo · Universidad EIA

Una PWA para documentar plantas del Bosque Andino durante recorridos de campo.
Diseñada para estudiantes de la Universidad EIA, inspirada en los diarios de campo de Alexander von Humboldt.


📱 ¿Qué es esto?
Una aplicación web progresiva (PWA) optimizada para móviles que permite a los estudiantes de la EIA generar fichas botánicas estéticas durante sus recorridos por el campus (2.150 m.s.n.m. · Bosque Andino), incluso sin conexión a internet.
✨ Características

📷 Captura o sube foto de la planta
🌳 Selector visual de tipo de planta (árbol, arbusto, hierba, trepadora)
🍂 Selector de forma de hoja con dibujos SVG
🎨 Paleta de colores para el tono predominante
✍️ Reflexión personal de campo
📥 Descarga la ficha como imagen PNG (lámina estilo siglo XIX)
📲 Instalable como app nativa ("Añadir a pantalla de inicio")
🌐 Funciona sin internet (Service Worker)
